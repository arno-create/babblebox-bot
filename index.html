Everyday Utilities

Useful Discord tools that try to stay dependable, low-noise, and privacy-conscious.

Watch V2

Mentions, replies, keywords

Separate mention and reply alerts, scoped keywords, ignored channels and users, and DM-only delivery with dedupe and cooldown protection.

Later

Resume the channel cleanly

Save one reading marker per channel, get a DM jump link back, and keep media-aware previews instead of awkward empty placeholders.

Capture

Private snapshots

DM yourself a recent-message transcript with better attachment summaries, no media blob storage, and no long-term archive table.

Moment / Remind / AFK

Small but memorable

Moment Cards create live-linked keepsakes, reminders stay bounded and one-time, and AFK handles `30m`, `2h`, `2d`, and compact combos like `1h30m`.

Daily Arcade

Babblebox Daily is a compact arcade instead of a single puzzle command.

Three Booths

Shuffle, Emoji, and Signal

Each UTC day generates the same shared board for everyone. Results stay compact and deterministic, but the presentation still feels rewarding and showable.

  • short booth runs with small attempt limits
  • shareable result output and leaderboard support
  • streak presentation tied back into Buddy/Profile
  • no rerolls and no external content APIs
Storage Discipline

Built for free-tier limits

Daily puzzle payloads regenerate from the date. Persistent rows only keep bounded result data, while streaks and lifetime totals stay in the profile row.

  • one result row per user per day per booth
  • raw rows pruned after 180 days
  • no image generation or heavy archives
  • low write volume and compact leaderboard reads

Buddy / Profile / Vault

Identity without turning Babblebox into an economy clone.

Buddy

Cosmetic-first charm

Each user gets one lightweight companion with a species, nickname, style, mood, title slot, and featured badge. Progress stays tied to real use, not grind loops.

Profile / Vault

One cohesive layer

`/profile` stays public-friendly and showable. `/vault` stays personal, pulling in live utility context without exposing private settings publicly.

Shield / Safety

Optional moderation support that stays compact, transparent, and conservative by design.

Local First

Privacy, promo, and heuristic scam packs

Shield can watch for likely privacy leaks, Discord invite or promo spam, and scam-style bait. Packs are configurable, log-first by default, and tuned through a private admin panel.

  • privacy leak pack
  • promo / invite pack
  • heuristic scam pack
  • scope controls, allowlists, trusted-role bypasses, and safe advanced wildcard patterns
AI Assist

Restricted second pass only

Shield AI assist never acts as the primary moderator. It only reviews messages that local Shield already flagged, only enriches moderator context, and never punishes on its own.

  • currently limited to guild 1322933864360050688
  • disabled outside that guild
  • sanitized and truncated content only
  • no stored AI-submitted message archive in Shield persistence

Privacy, Storage, and Support

Babblebox tries to stay useful without collecting more than it needs.

Storage Rules
  • Utility and profile persistence stay compact and bounded.
  • No media blob storage and no full watched-message archive.
  • Capture transcripts are delivered privately and not stored long-term.
  • Moment Cards are generated live from visible Discord messages and not stored as a quote feed.
  • Daily Arcade rows are compact and pruned after 180 days.
  • Shield stores config and safe pattern metadata, not a heavy moderation event archive.
